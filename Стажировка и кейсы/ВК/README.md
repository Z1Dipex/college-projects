# Кейс Влияние контентных эмбеддингов на рекомендации

## Описание
Задача состоит в том чтобы оценить чувствительность алгоритмов рекомендаций к коллаборативному и контентному сигналам. Все действия должны проводиться на выложенной модели нейросети. Данное задание оказалось сложным и непростым, и в данный момент не выполнено.
